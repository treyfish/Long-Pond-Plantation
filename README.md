# Long Pond Plantation

Multi-page marketing website for Long Pond Plantation, a wedding venue. Built by Cromis Digital as a static site (HTML + Tailwind CSS via CDN), deployed on Vercel.

## Local development

```bash
node serve.mjs
```

Serves the project root at http://localhost:3000.

## Screenshots

```bash
node screenshot.mjs http://localhost:3000 [label]
```

Saves auto-numbered PNGs to `temporary screenshots/` (requires `npm install puppeteer`).

## Brand assets

Logos, color guides, and photography live in `brand_assets/`.
